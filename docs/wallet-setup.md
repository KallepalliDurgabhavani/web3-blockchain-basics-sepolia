# MetaMask Wallet Setup

MetaMask is a Web3 wallet that allows users to interact with Ethereum-based applications.

## Steps to Set Up MetaMask
1. Visit https://metamask.io
2. Install the browser extension
3. Click "Create a Wallet"
4. Set a strong password
5. Backup the secret recovery phrase securely

## Importance of Wallet Security
- Never share your private key
- Store the recovery phrase offline
- Anyone with the phrase can access your wallet

## Result
After setup, the wallet is ready to connect to Ethereum networks and decentralized applications.
