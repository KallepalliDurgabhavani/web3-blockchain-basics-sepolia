# Web3 and Blockchain Basics – Sepolia Testnet

**Name:** Kallepalli Durga Bhavani  
**Network:** Ethereum Sepolia Testnet  

## Project Overview
This project demonstrates the basic concepts of Web3 and Blockchain by setting up a crypto wallet, configuring a test network, acquiring testnet ETH, and interacting with a decentralized application (DApp).

## Objectives
- Understand blockchain and Web3 fundamentals
- Set up MetaMask wallet
- Configure Ethereum Sepolia Testnet
- Connect to a DApp
- Use Infura RPC with Web3.js

## Technologies Used
- MetaMask Wallet
- Ethereum Sepolia Testnet
- Infura RPC
- Web3.js
- HTML & JavaScript

## How to Run
1. Install MetaMask browser extension
2. Configure Sepolia Testnet
3. Open `metamask-connect.html` in browser
4. Connect wallet and verify address
5. Run JavaScript files using Node.js
